# SPEEDY-CAN - SPEEDUINO 0.4 ADAPTER - RELEASE
A universal 35 pin adapter for the Speeduino 0.4 Ecu's


This is an easy to assemble pcb for the speeduino series ecu's.


The goal is to have an all in one pcb with the ability to have - 

A - CanBus output via esp32

B - Lamp outputs for features such as check engine light, Rpm limit, launch control illumination etc.

C - Avaliable inputs for oil/fuel pressure, vss etc.

D - Cheap to produce. (meet the 10x10 format) 

E - Easy to assemble using cheap existing modules and few components.


This can be used as a regular adapter without any need for moduleds or coding or logic.. it is simply there as an addition.


This is an early release, There is currently no coding avaliable. 

I have supplied information relative to pinouts required for your code.

There are existing speeduino canbus scripts that have useful information/code. (speeduino to realdash/serial2can etc) 


Treate it as an opensource community project. I have provided the bones.


UPDATE: after much testing, I have a prototype enclosure ready. 

Waiting for the 30-pin amp connector to confirm front alignment, but everything else has been fitment tested and is spot on. 

The enclosure has been made in a way where it's easy to print, and parts are replaceable or changes can be adapted without having to reprint an entire unit. 

2x side plates act as a jig to make sure the 40-pin connector is aligned correctly. 

The base has access to the esp32 boot and reset pins. Access to the USB port is there as well.
